Help

Migrating

  1. Log in via the Authorization page using your credentials for the CMS that contains the content you wish to migrate.
  2. Save a current snapshot by selecting "Snapshot" from the Tasks menu. See below for further help saving the snapshot.
  3. Now you are ready to migrate! Select "Migrate 1.1 to 1.2" from the Tasks menu and when prompted, log in to the CMS in which you want to upload your migrated version 1.2 content.
  4. Upon successful login, click the Start Migration button and migration will begin!

*Note, going to the Migration page will automatically log you out of the CMS you are logged in to. In order to perform any of the other functions in the app after visiting the Migration page, you will need to go to the Authorization page and log back in.

Saving a Snapshot

  1. Log in via the Authorization page using your credentials for the CMS that contains the content you wish to save.
  2. Select "Snapshot" from the Tasks menu.
  3. Click the "Choose" button and select an empty directory to save your snapshot into.
  4. Click "Save Snapshot". You will be prompted when the snapshot is done saving.
